# Smart Weather App

A modern, exclusive weather application built with a premium Glassmorphism design and real-time data integration.

## Features

### Implemented (Backend)
- **Real-time Weather Data**: Fetches accurate data via Open-Meteo API.
- **Global City Search**: Find weather for any city worldwide.
- **Smart Analysis**: Categorizes weather as Freezing, Cold, Cool, Warm, or Hot.
- **Clothing Suggestions**: Provides actionable advice on what to wear based on temperature and conditions (e.g., umbrella for rain).
- **Detailed Conditions**: Identifies specific weather states like Rain, Snow, Drizzle, and Fog.

### Planned (UI)
- Premium Glassmorphism design.
- Dynamic backgrounds based on weather conditions.
- Responsive and modern UI/UX.

## Usage
Run the script to check weather insights in the terminal:
```bash
python weather.py
```
