# Smart Weather App

A modern, exclusive weather application built with a premium Glassmorphism design and real-time data integration.

## Features (Planned)
- Real-time weather data via Open-Meteo API.
- Search functionality for cities worldwide.
- Smart weather state analysis (e.g., Freezing, Cold, Hot).
- Intelligent clothing suggestions based on temperature.
- Dynamic backgrounds based on weather conditions.
- Responsive and modern UI/UX.

## Backend Verification
To test the weather fetching logic:
```bash
python weather.py
```
